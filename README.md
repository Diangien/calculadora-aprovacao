*Calculadora de Aprovação Académica*
Esta calculadora é uma aplicação web desenvolvida para auxiliar estudantes universitários a planearem o seu desempenho académico.
Através das notas das Provas Parcelares (PP1 e PP2), o estudante pode calcular automaticamente:

* A nota mínima necessária no exame para obter aprovação;
* A sua classificação final após o exame;
* Se foi aprovado ou reprovado;
* Caso não seja aprovado no exame, a nota necessária no recurso;
* A classificação final após o recurso e a indicação de aprovação ou reprovação.

O aplicativo elimina a necessidade de realizar cálculos manuais, reduzindo erros e permitindo que o estudante tome decisões mais informadas sobre a sua preparação.
